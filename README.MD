## Краткое описание алгоритма решения ЗЛП

### Реализованный алгоритм решает задачу линейного программирования (ЗЛП) в общем виде, используя двухфазный симплекс-метод.
### Программа выполняет полный цикл решения, начиная с чтения входных данных и заканчивая выводом оптимального решения.

Этапы алгоритма:

1. Чтение задачи из файла
    
    Формат входного файла:

    ```txt
    max: 1 3 2 1
    c: 1 1 0 2 <= 8
    c: 0 1 1 1 = 6
    c: 2 0 1 0 >= 2
    ```
    Первая строка задает тип задачи (```max``` или ```min```) и коэффициенты целевой функции.
    
    Остальные строки описывают ограничения, начиная с префикса ```c:```

2. Приведение к канонической форме

    Все ограничения преобразуются в систему с равенствами путем добавления:

    1. дополнительных переменных для ```<=```;

    2. избыточных и искусственных переменных для ```>=```;

    3. искусственных переменных для ```=```.

    Формируется расширенная матрица ```A_ext```, вектор ```b_ext``` и расширенный вектор коэффициентов ```c_ext```.

3. Фаза 1 — решение вспомогательной задачи

    Цель — минимизировать сумму искусственных переменных.

    Если в конце искусственные переменные не равны нулю — исходная задача не имеет допустимого решения.

4. Фаза 2 — решение основной задачи

    Используется найденный базис из Фазы 1.

    Решается основная задача с исходной целевой функцией.

5. Вывод результата

    Оптимальные значения переменных ```x1, x2, ....```

    Значение целевой функции ```Z```.

    Проверка всех ограничений на корректность.
